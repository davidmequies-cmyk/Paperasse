# 📖 Mode d'emploi Hermès — comment déclencher chaque agent

Hermès se pilote **en langage naturel** (pas besoin de `/commande`). Dites simplement la chose ; le routeur choisit le bon agent.

## Phrases de déclenchement

**Pilotage**
- /dg — « Fais-moi le point de direction générale du jour. »
- /treso — « Fais-moi le point trésorerie / cash. »

**Paie & social**
- /paie — « Analyse ce dossier de paie intermittent : … »
- /rh — « Aide-moi sur ce sujet RH (salarié permanent) : … »

**Argent**
- /fiscal — « Analyse ce courrier fiscal / fais le point TVA. »
- /relance — « Rédige une relance pour cette facture impayée : … »
- /devis — « Prépare un devis pour : … »

**Juridique & contrats**
- /contrat — « Prépare un contrat de cession pour : … »
- /juridique — « Analyse ce dossier juridique / litige : … »
- /assur — « Fais le point sur cette assurance / ce sinistre : … »

**Projets**
- /immo — « Point immobilier sur l'offre Touques. »
- /prod — « Organise la production de cette prestation : … »
- /subv — « Quelles subventions pour ce projet ? »

**Visibilité**
- /seo — « Audite le SEO de paieintermittents.com. »
- /com — « Rédige un post pour annoncer : … »
- /artiste — « Demande de cérémonie pour David Méquiès : … »

**Admin**
- /assist — « Aide-moi sur cet administratif / agenda : … »

## Astuces
- Pour forcer un rôle précis : « **En tant qu'agent [Nom]**, … » puis coller le dossier.
- **Conseiller prud'homal** (votre fonction de juge) : ce n'est **pas** un agent Hermès — voir le dossier `prudhommes/` (sources officielles, méthode de délibération).

## Règle permanente
**Hermès prépare → vous validez ✅ / ✏️ / ❌ → il exécute.** Jamais de donnée inventée ; sources officielles pour les chiffres sensibles.
